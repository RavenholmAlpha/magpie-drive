# Magpie Drive 🐦

> 一个轻到不能再轻的Node.js文件管理系统，给不想搞复杂配置的你
> 整个项目其实就100kb
## 🎯 这是什么玩意儿？

Magpie Drive（喜鹊云）是一个用Node.js写的极简文件管理系统。
- 没有花里胡哨的界面
- 没有复杂的数据库配置
- 没有让人头大的依赖
- 只有最实用的文件管理功能

就像一只喜鹊，简单、快速，还能帮你搬东西（文件）。

## ✨ 老子有什么功能？

- 📤 **文件上传**：支持拖拽上传，批量上传，文件夹上传
- 📥 **文件下载**：单个下载，批量打包下载（Zip）
- 🗂️ **文件管理**：新建文件夹，移动文件，批量删除
- 🔍 **文件搜索**：全局搜索，快速定位文件
- 🔐 **用户认证**：支持admin和guest两种角色
- 📱 **响应式设计**：在任何设备上都能正常使用

## 🚀 怎么用？

### 1. 安装环境

首先，你得有Node.js（没有的话自己去下，别问我要）。

### 2. 启动服务器

```bash
# Linux/Mac
./start-server.sh

# Windows
start-server.bat
```

或者手动启动：

```bash
node server.js [端口号]  # 端口号可选，默认8000
```

### 3. 访问系统

打开浏览器，输入：`http://localhost:8000`

### 4. 登录

- **管理员账号**：admin / admin
- **访客账号**：guest / guest

## 🛠️ 技术栈

- **后端**：Node.js（纯原生，没使用任何框架，牛逼吧？）
- **前端**：HTML + CSS + JavaScript（同样纯原生）
- **其他**：JSZip（用于批量下载打包）

## 🤔 为什么写这个？

- 不想用那些臃肿的文件管理系统
- 不想配置数据库（SQLite都嫌麻烦）
- 想写一个自己能用的轻量级工具
- 想吐槽那些复杂的技术栈（没错，说的就是你，React + Express + MongoDB）
